#### 数学归纳法：用于证明断言对所有自然数都成立

- 证明过程
  - 证明断言对于 1 成立
  - 假设断言对于 n-1 成立
  - 证明断言对于 n 成立

- 以证明 1 + 2 + 3 + ... + n = n(1 + n) / 2 为例
  - 1 = 1 * (1 + 1) / 2 => 成立
  - 假设 1 + 2 + 3 + ... + (n - 1) = (n - 1)(1 + n - 1) / 2 成立，即假设 1 + 2 + 3 + ... + (n - 1) = n(n - 1) / 2
  - 则 1 + 2 + 3 + ... + n = [1 + 2 + 3 + ... + (n - 1)] + n = [n(n - 1) / 2] + n = n(1 + n) / 2；得证
